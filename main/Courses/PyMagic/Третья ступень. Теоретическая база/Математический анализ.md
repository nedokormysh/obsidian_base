# Числовая последовательность

Рассмотрим бесконечное упорядоченное множество вещественных чисел, элементы которого пронумерованы натуральными числами (индексами 𝑛 = 1, 2, 3, …):
${𝑥_n} = 𝑥_1, 𝑥_1,…, 𝑥_n,…$
Такое множество называется числовой̆ последовательностью, а его элементы
𝑥_i_ – членами числовой̆последовательности.

# Предел

Пусть дана последовательность {𝑥_n} и предположим, что числа с достаточно большим индексами из этой последовательности близки к числу 𝒂
Тогда, числовая последовательность имеет предел равный 𝑎:
$lim_{n→ \inf} = a$

(т.е. при каких-то индексов все числа последовательности лежат в эпсилон  окрестности а)

Число 𝒂 называется пределом числовой последовательности {𝑥_n} при 𝑛 → ∞, если для любого, сколь угодно малого, положительного числа 𝜀 найдется такое, достаточно большое, натуральное число N , что при 𝑛 > 𝑁 выполняется неравенство $|𝑥_𝑛 − 𝑎| < \epsilon$

𝜀 – число, характеризует разницу между числом 𝑎 и значением элемента последовательности с индексом 𝑛. Когда последовательность имеет предел a, то она сходится. Если нет, то
последовательность расходится.

где

## Производная сложной функции (chain rule)

Пусть 𝑦 = 𝑓(𝑥) дифференцируема в точке 𝑥_0, а функция 𝑧 = 𝑔(𝑦) дифференцируема в точке 𝑦_0 = 𝑓(𝑥_0). Тогда сложная функция 𝑧 = 𝑔(𝑓(𝑥)) дифференцируема в точке 𝑥_0, и ее производная равна:
$$𝑧'(𝑥_0) = 𝑔'(𝑦_0) 𝑓'(𝑥_0)$$
или
$$𝑧_x'= 𝑧_x'y_x'$$
или
$$\frac{𝑑𝑧}{𝑑𝑥}=\frac{𝑑𝑧}{𝑑𝑦}\frac{𝑑𝑦}{𝑑𝑥}$$
Это правило также распространяется на случай сложной функции (которая получена последовательными подстановками из произвольного числа дифференцируемых функций).
Например:
𝑢 = ℎ(𝑧), 𝑧 = 𝑔(𝑦), 𝑦 = 𝑓(𝑥)
$$\frac{𝑑𝑢}{𝑑𝑥}=\frac{𝑑𝑢}{𝑑𝑧}\frac{𝑑𝑧}{𝑑𝑦}\frac{𝑑𝑦}{𝑑𝑥}$$
## Частная производная
Частная производная—одно из обобщений понятия производной на случай
функции нескольких переменных. Частная производная —это предел
отношения 𝛥𝑦 по выбранной переменной к приращению этой переменной,
при стремлении этого приращения к нулю.

# Экстремум
Экстремум— это минимум или максимум функции.
Экстремумы бывают:
• Локальные (min/max не является наименьшим значением всей функции)
• Глобальные (min/max является наименьшим значением всей функции)

Необходимое условие экстремума— равенство нулю первой производной в данной точке, если такая производная существует. Данное условие не является достаточным. 

Первое достаточное условие локального экстремума
• Предположим, наша функция дифференцируема в точке 𝑥_0
• Непрерывна в точке 𝑥_0
• 𝑓′(𝑥_0) либо производной в этой точке не существует
• То при переходе через данную точку, производная меняет свой знак.
1) Если при 𝑥 < 𝑥_0 𝑓'(𝑥_0) > 0, а при 𝑥 > 𝑥_0 𝑓'(𝑥_0) < 0, то 𝑥_0 – точка максимума;
2) Если при 𝑥 < 𝑥_0 𝑓'(𝑥_0) < 0, а при 𝑥 > 𝑥_0 𝑓'(𝑥_0) > 0, то 𝑥_0 – точка минимума.

Второе достаточное условие локального экстремума
• 𝑦 = 𝑓(𝑥) имеет вторую непрерывную производную в окрестности 𝑥_0
• Первая производная в этой точке равна нулю 𝑓'(𝑥_0) = 0 и вторая производная в этой точке не равна нулю.
Тогда:
1) Если 𝑓''(𝑥) < 0, то 𝑥_0 – точка максимума
2) Если 𝑓''(𝑥) > 0, то 𝑥_0 – точка минимума

Итог: 
Знак первой производной влияет на рост функции:
• 𝑓'(𝑥) ≥ 0 – функция возрастает
• 𝑓'(𝑥) > 0 – функция строго возрастает
• 𝑓'(𝑥) ≤ 0 – функция убывает
• 𝑓'(𝑥) < 0 – функция строго убывает
Знак второй производной влияет на форму функции и выпуклость:
• 𝑓''(𝑥) ≥ 0 – выпукла вверх
• 𝑓''(𝑥) > 0 – строго выпукла вверх
• 𝑓''(𝑥) ≤ 0 – вогнута вниз
• 𝑓''(𝑥) < 0 – строго вогнута вниз

# Градиент

Градиент – это вектор из частных производных, указывающий направление наибольшего возрастания некоторой величины.
В следующем разделе более подробно будет рассмотрено понятие вектора, а пока можно просто представить как набор частных производных по 𝑛 переменным.
Формальное определение:
Если 𝑓(𝑥1,… , 𝑥𝑛 ) – функция 𝑛 переменных 𝑥1,… , 𝑥𝑛 , то
𝑛 – мерный вектор из частных производных:
$∇𝑓 =\Big(\frac{𝜕𝑓}{𝜕𝑥_1},…,\frac{𝜕𝑓}{𝜕𝑥}\Big)$

Градиент позволяет нам найти убывание или возрастание функции в той или иной точке.

В задачах оптимизации функции основная идея метода заключается в том, чтобы идти в направлении наискорейшего спуска – минимум ошибки модели. Наискорейшее убывание функции – это направление, которое задаётся антиградиентом −∇𝑓. Градиентом функции в точке называется направленный отрезок , отложенный от точки , который показывает
направление и скорость наискорейшего роста функции в данной точке.
